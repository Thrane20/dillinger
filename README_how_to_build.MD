# How To Build Dillinger Locally

## Prereqs

1. Install Docker

2. Install Rust/Cargo 
`curl https://sh.rustup.rs -sSf | bash -s -- -y`
Try running `cargo` after installing. If not found, do this:
`ENV PATH=~/.cargo/bin:${PATH}`
Then close and reopen your terminal for rust tools to be on the path.
